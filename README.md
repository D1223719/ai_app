# AI 履歷顧問機器人 (AI Resume Consultant)

這是一個基於 FastAPI、SQLite 與原生前端技術打造的對話式 AI 應用，專門用來協助求職者針對特定職缺 (JD) 客製化與優化履歷。

## 💡 核心功能 (Features)

1. **多專案管理 (Multi-Session Management)**
   - **說明**：找工作通常會投遞多間公司，系統左側的側邊欄允許您建立無限多個獨立的「聊天室 (Session)」。
   - **應用**：您可以為「Google 前端工程師」開一個專案，再為「Line 產品經理」開另一個專案。各專案的對話紀錄與履歷版本互不干擾。

2. **多模態文件分析 (Document & Image Parsing)**
   - **說明**：支援上傳您的履歷檔案或職缺描述 (JD)。得益於 Gemini 模型，甚至支援直接上傳截圖。
   - **應用**：免去繁瑣的「複製貼上」動作，直接把公司的 JD 網頁截圖上傳，並附上你的 PDF 履歷，讓 AI 自動去比對。

3. **具備記憶的對話式精修 (Conversational Refinement)**
   - **說明**：AI 會記住該 Session 內您上傳過的檔案與過去的對話歷史。
   - **應用**：不要指望 AI 一次就產出完美履歷。您可以像和真人顧問對話一樣，逐步提出要求，例如：「幫我把上一段的經歷縮短一點」、「語氣再自信一些」、「加上我帶領 5 人團隊的數據」。

4. **專業的獵頭人設 (System Prompting)**
   - **說明**：底層已經透過 `SKILL.md` 為 AI 注入了「資深獵頭」的人格設定，並強制要求它使用 STAR 原則 (情境、任務、行動、結果) 來改寫履歷。
   - **應用**：您不需自己構思複雜的 Prompt，只要直白地說出您的經歷，AI 就會自動以最專業的商業用語幫您包裝。

## 🚀 該如何開始應用？(User Workflow)

目前的終端機已經成功啟動了伺服器 (`Uvicorn running on http://127.0.0.1:8000`)。請按照以下步驟體驗：

### Step 1: 打開介面
打開瀏覽器，前往 [http://localhost:8000](http://localhost:8000)。您會看到一個類似 ChatGPT 的深色專業介面。

### Step 2: 建立目標
點擊左上方 **「+ 新增履歷諮詢」**，輸入您要應徵的公司與職位（例如："Amazon 資料科學家"）。

### Step 3: 上傳素材
在右側的主視窗上方，點擊 **「迴紋針圖示」**。
1. 上傳您的基礎履歷 (輸入檔案類型為 `resume_base`)。
2. 再次點擊迴紋針，上傳目標職缺的截圖或文件 (輸入檔案類型為 `job_description`)。

### Step 4: 指派任務
在下方的對話框輸入您的第一道指令，例如：
> *"我已經上傳了我的履歷和這份工作的 JD。請先幫我進行「差距分析」，告訴我有哪些經歷是契合的，哪些經歷需要補強？"*

### Step 5: 針對細節反覆打磨
收到 AI 的分析後，您可以針對某段經歷繼續優化：
> *"請幫我把履歷中『負責維護舊有系統』那段，根據這份 JD 強調的『效能優化』特質，重新用 STAR 原則改寫成 3 個重點。"*

## 技術架構 (Tech Stack)
- Frontend: HTML5 / CSS3 / Vanilla JavaScript
- Backend: Python FastAPI
- Database: SQLite (SQLAlchemy ORM)
- AI Model: Google Gemini 2.5 Pro
